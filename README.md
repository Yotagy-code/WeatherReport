☀️ Weather App

Описание

Этот проект — простое и элегантное приложение для прогноза погоды, которое предоставляет пользователю текущие данные о погоде, а также прогноз на несколько дней вперед. Приложение использует данные из OpenWeatherMap API, чтобы отображать текущую температуру, влажность, осадки, скорость ветра и другие параметры. Прогноз погоды отображается на основе данных, полученных для выбранного города или через определение местоположения пользователя.

Функционал

Текущий прогноз: Отображает текущую погоду, включая температуру, состояние, влажность, осадки и скорость ветра.

Прогноз на 4 дня: Показывает прогноз на ближайшие несколько дней.

Интерфейс: Современный и адаптивный дизайн, оптимизированный для разных устройств, включая мобильные телефоны, планшеты и десктопы.

Изменение местоположения: Возможность изменять город для получения прогноза для другого местоположения.

Использование геолокации: При отсутствии сохраненного города приложение использует IP-адрес для определения города или запрашивает местоположение пользователя через браузер.

Используемые API

OpenWeatherMap API

Приложение делает запросы к API для получения данных о текущей погоде и прогнозе на 5 дней. Вам нужно будет создать учетную запись на OpenWeatherMap и получить API-ключ для работы с данными. В проекте используется API-ключ:

const API_KEY = 'f14a962b738ce850ae61fa2e34a19f98';



Замените его на свой собственный ключ, если хотите использовать проект на своем сервере.

IP Geolocation API (ipapi)

Для определения местоположения пользователя через IP-адрес используется ipapi API. Это позволяет автоматически определить город пользователя, если доступ к его геолокации невозможен.

Описание функционала

Геолокация и поиск города

Приложение автоматически определяет местоположение пользователя с помощью его IP-адреса или браузерной геолокации. Если геолокация невозможна, оно использует "Bishkek" (Киргизия) как запасной город. Пользователь также может вручную ввести название города для получения прогноза.

Отображение данных о погоде

На главной странице отображаются следующие данные:

Текущая дата и день недели.

Город и страна.

Основная информация о погоде: температура, состояние (ясно, дождь, облачно и т. д.), осадки, влажность, ветер.

Прогноз на 4 дня: иконки с погодой и температура для каждого дня.

Изменение города

При нажатии на кнопку "Change Location" открывается модальное окно для ввода нового города. После ввода и подтверждения нового города, приложение обновляет информацию о погоде для выбранного города.

Адаптивность

Приложение полностью адаптивно для различных устройств:

Для мобильных устройств: Стили оптимизированы для экранов размером до 480px (например, для iPhone SE).

Для планшетов: Стили для устройств с экранами от 600px до 820px.

Для десктопов и больших экранов: Поддержка экранов шириной до 1366px, с адаптивной сеткой для прогнозов.

Пример интерфейса

Главная Карточка Погоды: Показывает текущий день, дату и погоду (иконка и температура).

Карточка Прогноза: Показывает прогноз на 4 дня с температурой и иконкой погоды для каждого дня.

Модальное Окно: Позволяет пользователю вводить название города и получать прогноз для нового местоположения.
